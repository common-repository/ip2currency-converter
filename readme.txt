Add this widget at your sidebar to display a currency converter.

== Description ==

IP2Currency Converter provides you a nice and ready-to-use widget to easily add the currency exchange rate function into your website. Below are the features:
- Support up to 58 commonly used currencies
- Daily updated exchange rate
- Customizable style, e.g, the color and skin.

Note: Your will need to register a free API key to gain access as the convertion results are calculated on IP2Currency server.

== Installation ==

*Using the WordPress dashboard*

1. Login to your wordpress
1. Go to Plugins
1. Select Add New
1. Search for IP2CurrencyConverter
1. Select Install
1. Select Install Now
1. Select Activate Plugin
1. Go to Widgets
1. Drag IP2CurrencyConverter to sidebar
1. In order to start query, generate a free license key from http://www.fraudlabs.com/freelicense.aspx?PackageID=10 and save your API key at settings page.

*Manual*

1. Download and unzip the plugin
1. Upload the entire *IP2CurrencyConverter/* directory to the */wp-content/plugins/* directory
1. Activate the plugin through the Plugins menu in WordPress
1. Go to Widgets
1. Drag IP2CurrencyConverter to sidebar
1. In order to start query, generate a free license key from http://www.fraudlabs.com/freelicense.aspx?PackageID=10 and save your API key at settings page.
